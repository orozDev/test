----------- ЗАПУСК ПРОЕКТА -----------------------

!Инструкция для первого запуска проекта

    1. Установите python для windows

    2. Откройте командную строку

    3. Перейти в папку проекта test в командной строке (используя cd)

    4. Пропишите команду для создание виртуального окружение: ```python -m venv venv```

    5. Далее пропишите команду для запуска виртуального окружение: venv\Scripts\activate.bat

    6. Далее пропишите команду для установки пакетов: pip install -r requirements.txt

    7. Далее пропишите команду для создании миграции: python manage.py makemigrations

    8. Далее пропишите команду для миграции: python manage.py migrate

    9. Далее пропишите команду для создания суперпользователя: python manage.py createsuperuser

    10. Далее пропишите команду для запуска проекта: python manage.py runserver


!Инструкция для запуска проекта ( После установки python и зависимых пакетов)

    1. Откройте командную строку

    3. Далее пропишите команду для запуска виртуального окружение: venv\Scripts\activate.bat

    4. Далее пропишите команду для запуска проекта: python manage.py runserver
