----------- ЗАПУСК ПРОЕКТА -----------------------

!Инструкция для первого запуска проекта

    1. Установите python для mac os

    2. Откройте командную строку

    3. Перейти в папку проекта test в терминале (используя cd)

    4. Пропишите команду для создание виртуального окружение: python3 -m venv venv

    5. Далее пропишите команду для запуска виртуального окружение: source venv/bin/activate

    6. Далее пропишите команду для установки пакетов: pip3 install -r requirements.txt

    7. Далее пропишите команду для создании миграции: python3 manage.py makemigrations

    8. Далее пропишите команду для миграции: python3 manage.py migrate

    9. Далее пропишите команду для создания суперпользователя: python3 manage.py createsuperuser

    10. Далее пропишите команду для запуска проекта: python3 manage.py runserver


!Инструкция для запуска проекта ( После установки python и зависимых пакетов)

    1. Откройте терминал

    2. Перейти в папку проекта test в терминале (используя cd)

    3. Далее пропишите команду для запуска виртуального окружение: venv/bin/activate

    4. Далее пропишите команду для запуска проекта: python3 manage.py runserver
